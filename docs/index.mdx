# Overview

`@nathan3boss/ui` is a React component library focused on practical product UI:
form controls, content surfaces, interaction patterns, and a theme layer that
can be scoped to part of an app.

## Start Here

- New to the library: go to [Get Started](/get-started.html) for installation and import patterns.
- Building forms: start with [Input](/components/input.html), [Textarea](/components/textarea.html), [Select](/components/select.html), [Radio](/components/radio.html), and [Switch](/components/switch.html).
- Building app chrome and content: see [Button](/components/button.html), [Badge](/components/badge.html), [Card](/components/card.html), [Tabs](/components/tabs.html), and [Tooltip](/components/tooltip.html).
- Setting up theming: visit [Theme](/components/theme.html) for `UiThemeProvider`, dark mode, and custom theme registration.
- Importing the library: use the subpath exports documented in [Get Started](/get-started.html).

## What You Get

- Accessible primitives built on native HTML semantics where possible.
- Composition-friendly APIs instead of large all-in-one components.
- Semantic token styling that works with app-level overrides and custom themes.
- ESM-first package exports for modern React environments.
- Generated API references for components, theme APIs, and public types.

## Public Exports

- Styles: `@nathan3boss/ui/styles` or `@nathan3boss/ui/styles.css`
- Actions: `@nathan3boss/ui/button`, `@nathan3boss/ui/badge`, `@nathan3boss/ui/spinner`
- Forms: `@nathan3boss/ui/input`, `@nathan3boss/ui/textarea`, `@nathan3boss/ui/select`, `@nathan3boss/ui/label`, `@nathan3boss/ui/helper-text`
- Selection: `@nathan3boss/ui/radio`, `@nathan3boss/ui/switch`, `@nathan3boss/ui/tabs`
- Content: `@nathan3boss/ui/card`, `@nathan3boss/ui/typography`, `@nathan3boss/ui/tooltip`
- Theming: `@nathan3boss/ui/theme`

## API Reference

Generated API docs are available under [/api](/api.html) and on each component
page. They cover the published package exports such as `Button`, `Card`,
`UiThemeProvider`, and the public theme contracts.
